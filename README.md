# Simple_video_platform

## 视频分享平台

![License](https://img.shields.io/badge/license-BSD--3--Clause-blue.svg)
![Python](https://img.shields.io/badge/python-3.6%2B-blue)

一个简单高效的基于 Flask 开发的视频分享平台！用户可以轻松注册、登录、上传和观看视频。

## 🎯 功能特性

- ✔️ 用户注册和登录
- ✔️ 视频上传与播放
- ✔️ 用户主页展示视频列表
- ✔️ 用户间的模糊搜索功能
- ✔️ 视频删除功能

## 🚀 快速开始
